Experience

  • I worked at Google as a Software Engineering Intern from May 2016 to Aug 2016. There, I worked on the Skia graphics engine, where I developed modern C++ code to emit GLSL shaders that render gradients with hard stops on the GPU more accurately and efficiently. As a result, over half of all hard stop gradients now render accurately with no loss in performance on all platforms that utilize Skia for 2D rendering (Chrome, Firefox, Android, etc.).
  • I worked at IBM as an Extreme Blue Technical Intern from May 2015 to Aug 2015. I worked on implementing a POSIX-compliant FUSE file system interface for Openstack Swift object storage in C and C++. I also travelled to IBM's HQ in Armonk, NY to pitch the project to senior executives.
  • I worked at ARA as a Technical Intern from July 2014 to Aug 2014. I contributed heavily to the Java frontend of BioGears, an open-source human physiology engine. I also translated key components of the core engine from C++ to Java for use in the frontend GUI.
